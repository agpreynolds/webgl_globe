| Installation Instructions |
-----------------------------
-----------------------------
The application is most effective when run from a locally hosted webserver. The instructions below assume you have such a server currently installed on your machine.
If you wish to acquire one:
http://www.apachefriends.org/en/xampp.html will provide you with instructions for how to set one up.

If you have a webserver running correctly:
1) Copy all files on the CD to the document root of your webserver
2) Run index.html in any of the following browsers:
	- Mozilla Firefox v25 or later
	- Google Chrome v31 or later
	- Chromium v30 or later

Otherwise run index.html in the following browser:
	- Mozilla Firefox v25 or later
